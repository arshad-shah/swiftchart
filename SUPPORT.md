# Support

Need help with SwiftChart? Pick the channel that matches your question.

## Documentation

The full documentation site is at <https://swiftchart.arshadshah.com>:

- **Getting started** for installation and quickstart.
- **Guides** for data mapping, theming, animations, responsive behaviour,
  and React refs.
- **Charts** for a per-chart-type reference with live previews.
- **API reference** auto-generated from TypeScript for both the core and
  React entry points.

## Questions and discussion

For "how do I", "is this possible", and architectural questions, open a
**GitHub Discussion** at
<https://github.com/arshad-shah/swiftchart/discussions>.

Discussions are searchable, so future users with the same question benefit
from your conversation.

## Bug reports

If something is genuinely broken, file a **bug report** at
<https://github.com/arshad-shah/swiftchart/issues/new?template=bug_report.yml>.
Please include the smallest reproducible example you can.

## Feature requests

Use the **feature request** template at
<https://github.com/arshad-shah/swiftchart/issues/new?template=feature_request.yml>.

Be specific about the use case. "I want X because Y" lands better than
"please add X".

## Security

Do **not** report security issues in public issues or discussions. See
[SECURITY.md](./SECURITY.md) for the private disclosure process.

## Commercial support

None offered at this time. SwiftChart is MIT-licensed and maintained on a
best-effort basis.

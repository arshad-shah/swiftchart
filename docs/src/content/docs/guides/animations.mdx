Animations are on by default. Three options control them:

| Option | Default | Effect |
|---|---|---|
| `animate` | `true` | Set to `false` to disable all animation. |
| `animDuration` | `600` | Duration in milliseconds. |
| `animEasing` | `'easeOutCubic'` | Easing curve from `EASINGS`. |

<ChartPreview
  client:visible
  kind="bar"
  data={monthlySales}
  mapping={{ x: 'month', y: 'revenue' }}
  options={{ theme: 'midnight', animDuration: 1200, animEasing: 'easeOutBack' }}
  height={300}
/>

The full list of easing names lives in `EASINGS`. All five:

- `linear` — uniform speed
- `easeOutCubic` — gentle deceleration (default)
- `easeInOutQuart` — slow start and end
- `easeOutBack` — slight overshoot
- `easeOutElastic` — playful bounce

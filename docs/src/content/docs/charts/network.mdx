## Tuning the simulation

Control the springs and repulsion via the chart config — the underlying solver
is exposed as a standalone helper too.

```ts
new NetworkChart('#chart', {
  iterations: 300,
  linkStrength: 0.08,
  chargeStrength: 400,
});
```

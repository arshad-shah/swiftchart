## Rose / Coxcomb / Nightingale

Set `rose: true` and the wedge *radius* (rather than its inset) encodes value.

<ChartPreview client:visible kind="radial-bar" data={traffic} mapping={{ labelField: 'source', valueField: 'visits' }} options={{ theme: 'arctic', rose: true }} height={300} />

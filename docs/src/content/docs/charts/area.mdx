Area is a line chart with `area: true`. Use the dedicated `Area` component in React, or pass `area: true` to `LineChart` directly.

<ChartPreview
  client:visible
  kind="area"
  data={monthlySales}
  mapping={{ x: 'month', y: 'revenue' }}
  options={{ theme: 'midnight' }}
  height={320}
/>

<Tabs syncKey="api">
  <TabItem label="Vanilla JS">
    ```ts
    import { LineChart } from '@arshad-shah/swift-chart';

    const chart = new LineChart('#chart', { theme: 'midnight', area: true });
    chart.setData(monthlySales, { x: 'month', y: 'revenue' });
    ```
  </TabItem>
  <TabItem label="React">
    ```tsx
    import { Area } from '@arshad-shah/swift-chart/react';

    <Area
      data={monthlySales}
      mapping={{ x: 'month', y: 'revenue' }}
      theme="midnight"
      height={320}
    />
    ```
  </TabItem>
</Tabs>

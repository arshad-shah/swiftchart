## Custom colour ramp

Pass a two-stop ramp via `colorScale`. Values interpolate linearly between the
stops.

<ChartPreview client:visible kind="heatmap" data={trafficGrid} mapping={{ x: 'hour', y: 'day', valueField: 'visits' }} options={{ theme: 'arctic', colorScale: ['#eef2ff', '#4338ca'], showValues: false }} height={300} />

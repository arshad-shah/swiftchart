## Percent (100 %-stacked)

Pass `percent: true` to normalise every column to 100 % so each segment shows
its share of the total instead of an absolute value.

<ChartPreview client:visible kind="stacked-bar" data={stacked} mapping={{ x: 'day', y: ['api', 'web', 'mobile'] }} options={{ theme: 'arctic', percent: true }} height={260} />
